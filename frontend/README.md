# Frontend - Gestao de Salas

Aplicacao Angular Material para:

- selecionar localidade (`WTorre` ou `Allianz`);
- listar salas em cards com status `Livre/Ocupado`;
- agendar sala com pre-validacao de conflito.

## Executar localmente

```bash
npm install
npm start
```

Frontend em `http://localhost:4200`.

## Build

```bash
npm run build
```
